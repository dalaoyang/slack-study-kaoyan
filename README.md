# 摸鱼考研神器 Demo

## 项目简介
本项目是一个基于智谱AI GLM4.5模型生成的"摸鱼考研神器"演示版本，包含2个HTML页面。工具专为考研学子设计，通过目标管理+专注学习的组合功能，帮助用户利用碎片时间设定学习目标并坚持完成，实现高效的"摸鱼式"复习。

## 核心依赖与技术来源
本项目的核心代码（包括页面结构、交互逻辑）主要通过以下资源生成：
- **基础模型**：[智谱GLM4.5](https://github.com/zai-org/GLM-4.5)  
  感谢智谱AI团队开源的GLM4.5模型，本项目的HTML页面代码均基于该模型生成，并根据实际需求进行了适当调整与优化。

## 页面文件说明
1. `goal-setting.html`（目标设定页）  
   功能说明：用于创建学习目标的页面，支持设置：
   - 学习时长（可选择5-60分钟的碎片时间）
   - 目标内容（可输入具体知识点，如"背诵10个考研英语单词"）
   - 学科分类（提供考研常见学科选项）
   - 提交后自动跳转至学习页面

2. `study-page.html`（学习页面）  
   功能说明：专注学习界面，核心功能包括：
   - 显示当前目标及剩余学习时间
   - 提供简约计时器（支持暂停但不支持提前结束）
   - 若尝试提前退出，会弹出提示"目标尚未完成，是否放弃？"
   - 完成目标后显示鼓励语及本次学习统计

## 使用方法
1. 下载本项目所有文件到本地
2. 直接用浏览器打开`goal-setting.html`即可开始使用：
   - 第一步：填写学习目标和时长并提交
   - 第二步：在学习页面专注完成设定内容
   - 无需安装任何额外依赖，数据暂存于浏览器本地

## 功能特色
- 目标约束：退出提示功能减少半途而废，提高学习执行力
- 时间灵活：支持短至5分钟的碎片学习，适配"摸鱼"场景
- 极简设计：界面简洁无干扰，降低在公共场景使用的突兀感
- 轻量便捷：无需注册登录，打开即可用，保护用户隐私
